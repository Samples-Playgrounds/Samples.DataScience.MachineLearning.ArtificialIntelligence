Automated questionnaires can include a wide variety of question types, each serving different purposes and data collection needs. Here are the main categories:

**Closed-ended questions:**
- Multiple choice (single or multiple selections)
- Yes/No or True/False questions
- Rating scales (1-5, 1-10, Likert scales)
- Ranking questions (order preferences)
- Dropdown menus
- Checkbox lists

**Open-ended questions:**
- Short text responses
- Long-form text areas
- Essay-style questions

**Specialized question types:**
- Matrix/grid questions (rate multiple items on same scale)
- Slider scales for continuous ratings
- Image selection questions
- File upload questions
- Demographic questions (age, gender, location, etc.)
- Contact information forms
- Date/time pickers

**Logic-based questions:**
- Conditional/branching questions (skip logic)
- Piped text questions (reference previous answers)
- Randomized question order
- A/B testing variations

**Common applications include:**
- Customer satisfaction surveys
- Market research
- Employee feedback
- Academic research
- Lead generation forms
- Registration forms
- Assessment and quiz questions
- Event feedback
- Product reviews
